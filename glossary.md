# 用語集

読みが特殊な用語や記号の意味とその読みをまとめています.

| 記述  | 意味                         | 読みカナ               |
| ----- | ---------------------------- | ---------------------- |
| `mut` | 変数に可変の指定をする       | ミュータ\|ミュータブル |
| `!`   | 処理が戻らない式の型         | ネバーガタ             |
| `()`  | 値が空の型                   | ユニットガタ           |
| `..`  | 数値の範囲でイテレータを作る | ハンイ                 |
